About the Project

AWS EC2 + IAM Setup

🖥️

EC2 Instance

Launched a t2.micro EC2 instance on AWS running Ubuntu/Amazon Linux 2, configured with a public security group allowing HTTP (port 80) and SSH (port 22) traffic.

🌐

Apache2 Web Server

Installed and configured Apache2 to serve this static website. Files are hosted at /var/www/html/ and served over port 80 to the internet.

🔒

Elastic IP

Allocated a static Elastic IP address from AWS and associated it with the EC2 instance to ensure a consistent public IP that doesn't change on restart.

👤

IAM User 1 — No Access

Created an IAM user with no policies attached. This user cannot view or interact with any AWS services including EC2 instances.

👥

IAM User 2 — EC2 Access

Created an IAM user with AmazonEC2ReadOnlyAccess policy. This user can view and describe EC2 instances, security groups, and related resources.

📋

Assignment Goal

Demonstrate cloud infrastructure skills including compute provisioning, web serving, static IP management, and identity/access management on AWS.
