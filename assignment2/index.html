About the Project

AWS S3 + Versioning Setup

🪣

S3 Bucket Creation

Created a globally unique Amazon S3 bucket to store all the static assets (HTML, CSS, JS) required for this website.

🌐

Static Website Hosting

Enabled the Static Website Hosting feature on the S3 bucket, configuring `index.html` as the default entry point and setting up public bucket policies.

⏱️

Bucket Versioning

Enabled S3 Versioning to preserve, retrieve, and restore every version of every object stored in the bucket, protecting against accidental overwrites.

♻️

Lifecycle Rules

Implemented lifecycle management rules to automatically transition or permanently delete noncurrent versions of objects after 30 days to optimize storage costs.
